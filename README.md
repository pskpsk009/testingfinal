# SE Project Hub

This repository currently contains two frontend folders.

## Canonical Frontend

Use `frontend/` as the main frontend application.

- Active app for ongoing development
- Target for all new UI features and bug fixes
- Recommended Render frontend Root Directory: `frontend`

## Legacy Frontend

`backend/client/` is a legacy frontend copy kept for reference.

- Do not use it as the primary deployment target
- Do not add new product features there unless explicitly required

## Deployment Guidance

- Frontend service Root Directory: `frontend`
- Backend API Root Directory: `backend/api`
